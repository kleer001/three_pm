# 3pm — Period & Cringe Lint

Banned-list for the BUDDY messenger copy. The Period editor and the pre-draft
check this. Date-stamped against real etymology research.

Setting: **200X** — the game's era is "200X" (a Mega Man-style placeholder), so
**all of 2000–2009 is fair game.** This is looser than a strict 2002 setting:
slang that emerged 2003–2009 (`xD`, `idk`, `omw`, `smh`, `lmfao`, `tbh`) is
period-accurate here. Only ban what's genuinely *post-2009*, plus smartphone-era
tech that clashes with the dial-up / MSN-era look.

---

## ALLOW — period-accurate (2000–2009)

Free to use, in-character and sparingly (remember the <3% rule in `voices.md`):

- **Chat shorthand:** brb, bbl, bbs, g2g, gtg, ttyl, ttfn, afk, wb, np, omw, idk,
  asl / a/s/l, l8r, cya, jk, thx, plz, u, ur, r, cuz, ne1, sup, irl, btw.
- **Reactions:** lol, lmao, lmfao, rofl, omg, wtf, smh, tbh, haha, hehe.
- **Gamer/leet:** owned, pwned, n00b/newbie, w00t, 1337/leet/l33t, gg, pr0n, brt.
- **Emoticons (ASCII only):** `:) :( ;) :P :D =) :/ </3 <3 <333` and kaomoji
  `^^ o_O o.0 xD ^_^ -_- >_<`.
- **MSN emoticons:** simple typed smileys render as little graphics in this era.
  Fine. (Animated/custom MSN emoticons are 2003+ — also inside the 200X window.)

Note: `pwned` (the "p" typo of "owned") only appears ~2001, so it reads as
*brand new* early in the decade — that's a feature, not a bug.

## BAN — smartphone / social-media era (breaks the dial-up aesthetic)

Even where technically inside 2000–2009, these clash with the BUDDY/MSN/Windows
setting. Don't reference:

- **Devices/apps:** iPhone, smartphone, Discord, TikTok, Snapchat, Instagram,
  Facebook (post-2004 anyway), Zoom, "the cloud."
- **selfie** — one obscure 2002 citation, but zero currency until ~2012. Reads
  anachronistic. Ban unless a deliberate deep-cut.
- **Graphical Unicode emoji** (😂 ❤️ 🙂 💀) — phone-era (2011+). Use ASCII
  emoticons instead. *(The mock UI may show a system emoji as an icon stand-in;
  that's placeholder art, not copy. Copy stays ASCII.)*

Period-correct tech is fine and encouraged: walkman/discman, AIM/MSN, dial-up,
burned CDs, LimeWire/Kazaa, flip phones with T9, gel pens, Tamagotchi.

## CRINGE — the authenticity failures (from the corpus)

Not date errors — *texture* errors that make copy read fake:

1. **Over-abbreviation.** Real teens wrote mostly normal English. Do **not**
   stack shorthand (`omg g2g ttyl l8r brb <3 xD` in one breath). One or two
   markers per line, max. If a line is all abbreviations, it's wrong.
2. **`lol` as laughter.** It's filler ("yeah/cool/I'm here"). Don't deploy it
   only at jokes.
3. **Whole-message CAPS** (except Chad, whose card allows it). CAPS is for
   *emphasis on a word*, not the sentence.
4. **Emoticon spam.** ~1 per exchange, not per line.
5. **Tidy apostrophes everywhere.** Inconsistency is realer than perfection.
6. **Modern cadence.** No "it's giving," no "the way that…", no Gen-Z syntax.
7. **Everyone sounding the same.** Each line must pass the Voice editor's
   name-stripped test (see `editors/voice.md`).
